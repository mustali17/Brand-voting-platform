# Prerequisites
- Node.js (>=18.17)

# Installation
1. Clone the repository


2. Install dependencies

```
cd folder_name
pnpm install
```

3. Configure environment variables
- In the root create .env file and configure the following variables

```
MONGO_URI=
GITHUB_ID=
GITHUB_SECRET=
NEXTAUTH_SECRET=
NEXTAUTH_URL=
GOOGLE_ID=
GOOGLE_SECRET=
```

4. Run the application

```
pnpm run dev
```
